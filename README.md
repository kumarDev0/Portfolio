# Deepak Kumar — Portfolio Website

## 🚀 Features
- Premium dark theme with neon accents
- Custom animated cursor
- Smooth parallax & scroll animations
- Typed text effect (hero section)
- Animated counters
- 3D tilt effect on cards
- AI Chatbot (knows everything about Deepak)
- Resume download button
- Contact form
- Fully responsive (mobile-friendly)

## 📁 Project Structure
```
portfolio/
├── index.html          ← Main HTML file
├── css/
│   └── style.css       ← All styles + animations
├── js/
│   └── main.js         ← All interactivity + AI chatbot
├── assets/
│   ├── Deepak_Kumar_Resume.docx  ← Resume (downloadable)
│   └── deepak.jpg      ← YOUR PHOTO (add this!)
└── README.md
```

## ⚡ How to Run Locally (VS Code)
1. Open the `portfolio` folder in VS Code
2. Install **Live Server** extension (if not installed)
3. Right-click `index.html` → **Open with Live Server**
4. Opens at `http://127.0.0.1:5500`

OR simply double-click `index.html` to open in browser.

## 📸 Add Your Photo
1. Add your photo file to the `assets/` folder
2. Name it exactly: `deepak.jpg`
3. Recommended: square crop, min 400×400px
4. If no photo is added, a stylish "DK" placeholder shows instead

## 🌐 Deploy to Vercel
1. Push this folder to a GitHub repository
2. Go to vercel.com → New Project
3. Import your GitHub repo
4. Click Deploy — done! ✅

No build step needed. Pure HTML/CSS/JS.

## 🎨 Customize
- Colors: Edit CSS variables in `css/style.css` (top of file)
- Chatbot: Edit `DEEPAK_INFO` object in `js/main.js`
- Projects: Edit the project cards in `index.html`
- Contact links: Update email, LinkedIn, GitHub links in `index.html`
